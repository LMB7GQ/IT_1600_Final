## What Is Markdown
Markdown is a formatting language that allows users to format plain text using symbols like ('#' pr '*') to stylize their content.

Markdown is mostly used for
- Documentation
- Github README files
- notes
- Technical guides

---
## Why use markdown
- it is easy to learn
- t is universal
- great tool for documentation
- makes text clean and easy to read
- it is easy to write up organized notes using only keyboard and not having to rely other tools to provide header, bold, tables and other tools
