## Text Formatting 

The different stylization that can be used in markdown aer

| Style | Markdown | Example |
|-------|-----------|---------|
| Bold | `**text**` | **Bold** |
| Italic | `*text*` | *Italic* |
| Bold + Italic | `***text***` | ***Bold Italic*** |
| Strikethrough | `~~text~~` | ~~Striked~~ |
| Highlight | `==text==` | ==Highlighted== |
